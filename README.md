# liquidity-event-listener

--------------------------------------
1. Faça o clone do repositório
--------------------------------------

--------------------------------------
2. Instale as dependências abaixo:

npm install
npm install ethers
npm install web3
npm install dotenv
--------------------------------------

--------------------------------------
3. Renomeie o arquivo .env.example
   para .env e altere as variáveis do arquivo
--------------------------------------

--------------------------------------
4. Rode o comando node index.js
--------------------------------------
